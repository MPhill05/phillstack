Mark Phillips


Creative Full-Stack web developer, as well as native iOS developer via Swift. I have roughly 3 years of self taught web experience building full responsive web applications with a focus on the sites being easy to navigate with a clean design. I also build robust mobile apps via Swift for iOS. Taking technical ideas and bringing them to life in a simple, elegant mobile app has always been a passion of mine.


Whether I'm building a mobile app or website, I like to build projects with the whole picture in mind. I avoid building things page by page, but try to build the project to feel like one unified experience. I build projects much like I write music, where elements of every page intertwine throughout my apps and sites. I like to make sites that feel familiar while being new and fresh.


With iOS I tend to build applications with Swift and utilize tools such as Firebase and Cocoapods. For my front end process I tend to build sites with Sass, Bootstrap and ANgular 5. I also work great in a team setting, utilizing tools such as Git and am agile in my development process, working closely with my clients and frequently getting new features out to them to find out what is and is not working as soon a possible. I use NPM to build all my sites and test for various devices with mobile and desktop.
